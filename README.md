# Finance Tracker

**[Open App](https://siriyellu.github.io/Finance-Tracker/)**

Live finance tracking platform with stock watchlist, portfolio tracker, wealth growth projections, and expense tracking.

## Features
- Live Stock Watchlist — real-time prices via Yahoo Finance
- Spin-off & Breakout Tracker — SNDK, NVDA, VOO, TSM, ANET
- Portfolio Tracker — holdings, P&L, allocation pie chart
- Wealth Builder — compound growth chart with sliders
- Expense Tracker — categories, budget, charts
- Market Overview — S&P 500, NASDAQ, DOW, VIX live
- Dark / Light mode

## Run Locally
```bash
npm install
npm run dev
```

## Deploy
```bash
npm run deploy
```
